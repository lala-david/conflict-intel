# Terror Researcher — Product Plan

## Mission

**"분쟁과 테러 데이터를 누구나 이해할 수 있게 만든다."**

전 세계 테러/분쟁 정보는 6개 이상의 데이터 소스에 흩어져 있고, 이를 통합 분석하려면 연간 $100K 이상의 엔터프라이즈 도구가 필요하다. 우리는 이 데이터를 자동으로 수집/정제/분석하여, 전문 분석관 수준의 인텔리전스를 누구에게나 제공한다.

---

## 1. 시장 분석

### 경쟁 구도

| 구간 | 서비스 | 가격 | 한계 |
|------|--------|------|------|
| **무료/학술** | ACLED, UCDP, GTD, Liveuamap | $0 | 실시간 아님, UX 열악, 교차검증 없음 |
| **중간** | **(비어있음)** | $50~500/월 | **아무도 없음** |
| **엔터프라이즈** | Janes($150K+), Recorded Future($100K+), SITE, Dataminr($500K+) | $100K~500K/년 | 기업/정부만 접근 가능 |

### 핵심 공백 (아무도 안 하는 것)

1. **실시간 + 구조화 데이터 통합** — ACLED은 구조화됐지만 1주 지연, GDELT는 실시간이지만 노이즈 투성이. 둘을 합친 것이 없음
2. **교차 소스 검증** — 같은 사건을 GDELT/UCDP/뉴스에서 교차 확인하는 시스템이 없음
3. **중간 가격대** — 무료와 $100K 사이에 전문가급 도구가 없음
4. **현대적 UX** — 학술 도구는 2010년대 UX, 엔터프라이즈는 폐쇄적
5. **예측 + 실시간 + 이력을 한 화면에** — 아무도 안 함
6. **개발자 친화적 API** — 합리적 가격의 테러 데이터 API가 없음

## 2. 우리의 차별점

### 이미 가진 것 (경쟁우위)

| 우리가 가진 것 | Janes/SITE에 있나? | ACLED/UCDP에 있나? | Liveuamap에 있나? |
|---------------|-------------------|-------------------|------------------|
| 6소스 자동 수집 + 병렬화 | O (수동 큐레이션) | X (단일 소스) | X (수동) |
| GDELT+UCDP+뉴스 교차매칭 | X | X | X |
| 286개 조직 + 42개 약어 자동 매칭 | O | X | X |
| 71개 분쟁지역 좌표 매칭 | O | 부분적 | O |
| 국가별 위협도 자동 계산 | O | X | X |
| 핫스팟 격자 분석 | O | X | X |
| 17개 전문가 RSS 통합 | X | X | X |
| 완전 오픈소스 | X | 부분적 | X |

### 우리만의 핵심 차별점

**1. Cross-Source Intelligence (교차 소스 인텔리전스)**
- 같은 사건을 GDELT(미디어) + UCDP(코딩된 사건) + 뉴스(텍스트)에서 자동 교차 검증
- "이 사건은 3개 소스에서 확인됨" → 신뢰도 자동 산출
- 경쟁사 중 이걸 자동화한 곳 없음

**2. Open & Affordable**
- 완전 오픈소스 — 누구나 자기 서버에서 돌릴 수 있음
- 호스팅 버전은 $50/월 수준으로 제공 가능 (LLM 비용 $1/월 수준)
- Janes의 1/3000 가격에 핵심 기능 제공

**3. Developer-First API**
- 테러/분쟁 데이터를 구조화된 JSON API로 제공
- 보안 회사, 여행 앱, 보험사가 자기 서비스에 연동 가능
- 현재 이런 API 서비스가 합리적 가격에 존재하지 않음

## 4. 제품 로드맵

### Phase 1: 대시보드 (4주)
> 지금 가진 데이터를 시각화

- 세계 지도 — 핫스팟 히트맵 (좌표 데이터 활용)
- 국가별 위협도 카드 (색상 코딩)
- 일일 BLUF 카드
- UCDP 사건 타임라인
- 조직 활동 차트
- 전일/주간 트렌드 그래프
- 기술: Next.js + Mapbox/Leaflet + D3.js + SQLite

### Phase 2: API + 알림 (4주)
> 다른 서비스에서 쓸 수 있게

- REST API (`/api/threats`, `/api/events`, `/api/orgs`)
- Telegram/Discord 봇 알림
  - "Nigeria 위협도 SEVERE 도달"
  - "TTP 활동 급증 (전일 대비 300%)"
  - "신규 제재 대상 3건"
- 국가/조직 구독 기능
- API 키 발급 + rate limiting

### Phase 3: 예측 + 심화 (8주)
> 과거 데이터로 미래 예측

- ViEWS 예측 데이터 통합 (1~36개월 분쟁 예측)
- 자체 트렌드 예측 (SQLite 누적 데이터 기반)
- 조직 네트워크 그래프 (동맹/적대 관계 시각화)
- 사건 상세 페이지 (교차 소스 증거 나열)

### Phase 4: 커뮤니티 + 수익화 (이후)
> 지속 가능한 모델

- 오픈소스 코어 + 호스팅 SaaS ($50~200/월)
- API 과금 (1,000 calls/일 무료, 이후 유료)
- 커스텀 보고서 생성기 (국가/조직/기간 선택)
- 커뮤니티 기여 (조직 DB, 분쟁지역 업데이트)

## 6. 사명과 목표

### Mission
**"분쟁 정보의 민주화 — 누구나 세계의 위협을 이해하고 대비할 수 있게 한다."**

### Vision
Janes가 정부에게 하는 일을, 우리는 모든 사람에게 한다.

### 핵심 가치
1. **투명성** — 모든 데이터의 출처를 명시, 코드는 오픈소스

4. **자동화** — 사람이 매일 할 일을 0으로
